    @set hello String "Hello AsmX!"; 
    
    @memory hello  0x01  # write to memory
    @address 0x01 txt # define address of the memory
    
    @Route  txt 0x01 # route
    @Stack  0x01  # stack set point with address
    @Invoke 0x04
    
    @Invoke 0x80 # call kernel

            1. Set variable
            This is a tutorial for those who want to learn the asmx programming language. Let's start simple.
            A stack is like a bank cabinet with cases and the data in them is replenished from the bottom up. 
            when an instruction is executed, the results can be pushed onto the stack.
            The register is like a keychain or a ticket by which we can access it and get something.
            Memory is a kind of closet with data that is stored for some time.
            Let's start with something simpler, namely creating a variable. 
            When a variable is created, the result is not written to memory. 
            We need to use the @set statement. Its syntax is:
            @set name <Type> value
            when accessing the register $arg0, you can find out the name of the variable.
            $arg1 - variable type
            $arg2 - variable value
            When this instruction is executed, the value of the variable will be written onto the stack.
            Variable assignment example:
            
                

                    @set txt String "Hello world!"
                

            
            By default, if you don't specify a type, it will be Int:
            
                

                    @set age 24 # default type Int
                

            
            Types that can be assigned to variables:
            String - string type
            Bool - boolean type that can contain true or false value
            Int - a numeric type containing an unsigned integer.
            Float - a numeric type containing an integer with a signed fractional part.
            List - Sips of cells, each of them contains its own data.
            An example of assigning a variable to all types:
            
                

                    @set age  14; #  default type Int
                    @set name String "E5war5IT"; 
                    @set isAge Bool true;
                    @set rating Float 4.5;
                    @set stars Int 450;
